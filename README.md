# 아르카나 아카데미 (Arcane Academy)

마법학교를 배경으로 한 **2D 사이드뷰 도트 액션 RPG** 프로토타입.
마법학교 아르피아의 성장·수업·종족·직업 감성에, 메이플스토리·산나비식 스타일리시한
전투와 타격감을 결합했습니다.

> 의존성 0, 빌드 0 — 단일 `index.html`(HTML5 Canvas + vanilla JS). 파일 하나가 게임 전부입니다.

## 플레이 방법

- **로컬:** `index.html`을 브라우저(크롬 권장)로 열기.
- **웹:** GitHub Pages를 켜면 `https://abback-go.github.io/yemogame/` 에서 바로 플레이.
  (저장소 **Settings → Pages → Branch: `main` / `/ (root)`** 로 설정)

## 조작

| 입력 | 동작 |
|---|---|
| ← → | 이동 |
| Space / ↑ | 점프 (↓+Space: 발판 아래로 내려가기) |
| Shift | 대시 (짧은 무적) |
| E | 대화 / 상호작용 · ↑ 포탈 이동 |
| Z | 불티 · X(홀드) 화염구 · C 화염 장판 |
| V | 플레임 랜스(관통) · S 폭염 노바(광역) |
| Q | (홀드) 인페르노 — 1식 · R 멸화 — 2식 |
| G | 유성 질주(돌진·균열 파괴·레일) · B 섬화(점멸) · F 도깨비불 |
| 이동술 | 공중 Space=이중점프 · 벽+Space=벽점프 · 낙하 중 Space 유지=활공 |
| Tab | 학생 수첩 (스탯 랭크·퀘스트·스킬 도감) · M 월드맵 · H 조작법 |

> **모든 마법은 교수님 수업으로 배워야 사용 가능** — 입학 직후엔 스킬이 하나도 없습니다.
> 교수에게 **E**로 말을 걸면 **학생 수첩식 수업 선택 창**이 열려, 배우고 싶은 술식을 골라 습득합니다.

## 아트 & UX (v0.20 리마스터)

"프로토타입"에서 "완성된 게임"의 룩으로 전면 폴리시:
- **타이틀 화면:** 전용 야경 씬(달·별·원경 학교 실루엣·마법 불티)·정제된 로고
- **분위기 렌더링:** 존별 컬러 그레이드·비네트·원경 안개·갓레이·앰비언트 파티클·별
- **지형 디테일:** 지면 그라데이션·상단 하이라이트·풀/눈 캡 등 유기적 디테일
- **HUD:** 랭크 뱃지·광택 게이지·아이콘, 통일된 오너먼트 패널(수첩·공방·월드맵·옵션)
- **안내판:** 큰 텍스트 박스 → 작은 팻말 + 근접 시 툴팁(화면 정리)
- **대화창:** 네임플레이트·초상 스와치·타이프라이터 연출
- **지역 진입:** 상단 지역명 타이틀 카드 + 부드러운 전환 페이드

## 게임 흐름 (v0.14 스토리 개편)

**54개 퀘스트의 메인 스토리 스파인**이 나침반 역할을 하고, 그 바깥의 **이어진 세계**를 자유롭게 탐험합니다:

1. **튜토리얼** — '학교 가는 길' 언덕에서 이동·점프·대시·아래점프를 배우고, 동급생 **리나**를 만나 입학
2. **입학 오리엔테이션** — 담임 **아그니스 교수**와 대면, 도서관·양호실·매점·훈련실·교장실 투어
3. **스킬 습득 아크** — 스킬 하나를 얻으려면 **이론 수업 → 심부름 퀘스트 → 과제 → 각인 실습**의 여러 단계를 거쳐야 함
   (예: 불티 = 이론 I → 교과서 찾기 → 테오의 노트 → 마력 측정 → 각인 실습 10회 적중)
4. **가벼운 학교 스토리** — 리나의 잃어버린 목걸이, 테오의 버섯 공포증, 교장의 소포 심부름, 잡화점 젤리 배달…
5. **초급(3식) → 학기말 실기 시험 → 중급(2식) → 고급(2식)** 순서로 성장

지금 말을 걸어야 하는 인물 머리 위에는 **금색 "!" 마커**가 표시됩니다.

## 현재 구현된 것 (프로토타입)

- **스토리 스파인:** 튜토리얼~고급 마법까지 54개 퀘스트 체인 (동급생·교사·마을 NPC 스토리)
- **전투/타격감:** 히트스톱·화면 흔들림·파티클·데미지 넘버·크리티컬·콤보
- **스탯 랭크 시스템:** 숫자 대신 D~SSS 랭크 (『퀘스트지상주의』 참고), 퀘스트·수업으로 수련
- **수업 선택 창:** 교수와 대화 시 책 형태 UI에서 원하는 술식을 골라 습득 (선행 조건은 '잠김' 사유 표시)
- **수업(스킬 습득):** 초급(담임 아그니스) → 중급(이그니아) → 고급(발더스 대마도사) 교사 티어
- **고급 마법 2식:** 제1식 「인페르노 번」(램프업 단일 광선) · 제2식 「멸화」(화면 전체 별똥별 광역)
- **대륙 탐험 (v0.16~19, 할로우 나이트식 메트로배니아):** 대형 존 **12개** — 아르카나 언덕/깊은 숲/지하 미궁/노을 해안/불의 마을 + **서리고개·얼음 마을·별빛 광산·모루골(대장간)·부유 정원·재의 거리·첫 불꽃의 신전**. 20여 개 연결점의 상호 순환 지도
- **시원한 이동감:** 코요테 타임·점프 버퍼·가변 점프·대시 모멘텀
- **이동 술식 6종(탐험으로 습득):** 이중점프·벽점프·도깨비불·유성 질주·활공·섬화 — 능력이 곧 열쇠(균열 암벽·빙벽·재의 장막·상승 기류 게이트)
- **존 고유 기믹:** 온기 게이지(설산)·유성 질주 레일(광산)·상승 기류(부유 정원)·재의 장막(재의 거리)
- **마도각인(부적):** 모루골 대장간에서 마력 결정으로 제작·장착 (3슬롯)
- **미니보스:** 학기말 실기 시험(숲의 수호자) · 불의 마을 화염 군주 이그니스
- **지하실:** 으스스한 미로·횃불·비네트·그림자 박쥐·의문의 문(미래 콘텐츠 훅)
- **자동 저장/이어하기** (localStorage) — 업데이트 후에도 기존 저장 데이터 유지 (구버전 세이브는 보유 스킬 기준으로 신규 퀘스트 체인 위치 자동 계산)
