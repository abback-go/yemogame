# 아르카나 아카데미 (Arcane Academy)

마법학교를 배경으로 한 **2D 사이드뷰 도트 액션 RPG** 프로토타입.
마법학교 아르피아의 성장·수업·종족·직업 감성에, 메이플스토리·산나비식 스타일리시한
전투와 타격감을 결합했습니다.

> 의존성 0, 빌드 0 — 단일 `index.html`(HTML5 Canvas + vanilla JS). 파일 하나가 게임 전부입니다.

## 플레이 방법

- **로컬:** `index.html`을 브라우저(크롬 권장)로 열기.
- **웹:** GitHub Pages를 켜면 `https://abback-go.github.io/yemogame/` 에서 바로 플레이.
  (저장소 **Settings → Pages → Branch: `main` / `/ (root)`** 로 설정)

## 조작

| 입력 | 동작 |
|---|---|
| ← → | 이동 |
| Space / ↑ | 점프 (↓+Space: 발판 아래로 내려가기) |
| Shift | 대시 (짧은 무적) |
| E | 대화 / 상호작용 · ↑ 포탈 이동 |
| Z | 불티 · X(홀드) 화염구 · C 화염 장판 |
| V | 플레임 랜스(관통) · S 폭염 노바(광역) |
| Tab | 학생 수첩 (스탯 랭크·퀘스트) · H 조작법 |

> **모든 마법은 교수님 수업으로 배워야 사용 가능** — 입학 직후엔 스킬이 하나도 없습니다.

## 현재 구현된 것 (프로토타입)

- **전투/타격감:** 히트스톱·화면 흔들림·파티클·데미지 넘버·크리티컬·콤보
- **스탯 랭크 시스템:** 숫자 대신 D~SSS 랭크 (『퀘스트지상주의』 참고), 퀘스트·수업으로 수련
- **수업(스킬 습득):** 초급(아그니스) → 중급(이그니아) → 고급(잠금) 교사 티어
- **다중 맵 + 포탈:** 캠퍼스(허브) ↔ 입문의 숲 ↔ 학교 지하실
- **미니보스:** 학기말 실기 시험(숲의 수호자) — windup 예고 + 2페이즈
- **지하실:** 으스스한 미로·횃불·비네트·그림자 박쥐·의문의 문(미래 콘텐츠 훅)
- **자동 저장/이어하기** (localStorage)
