## 0. 한 줄 요약 (Elevator Pitch)

> "마법학교에 입학한 신입생이 되어, **수업과 퀘스트로 마법을 배우고**, **메이플처럼 경쾌하고 스타일리시한 사이드뷰 전투**로 마물과 시험을 돌파하며, 종족·학과(직업)를 선택해 나만의 마법사를 키우는 도트 액션 RPG."

---

## 1. 디자인 필러 (Design Pillars)

기획의 모든 판단은 아래 4개 기둥으로 되돌아온다. 충돌 시 우선순위는 번호 순.

1. **손맛 있는 마법 전투 (Feel First)**
   메이플의 경쾌함 + 산나비의 묵직한 도트 타격감. 한 번의 스킬 시전에도 히트스톱·화면흔들림·파티클·이펙트가 반응한다. "마법을 쏘는 것 자체가 재밌어야 한다."

2. **배우며 강해지는 성장 (Learn to Grow)**
   아르피아식으로, 강함은 레벨업뿐 아니라 **수업을 듣고 퀘스트를 풀어 실제로 마법·스킬을 "습득"**하면서 온다. 성장 경로가 곧 스토리 진행 경로.

3. **정체성 커스터마이즈 (Be Someone)**
   종족 + 학과(직업) + 스킬트리 + 장비 조합으로 "내 캐릭터"를 만든다. 종족과 학과가 조작감·플레이스타일을 실제로 바꾼다.

4. **학교라는 세계 (A Living School)**
   기숙사·교실·식당·훈련장·도서관·금지구역이 하나의 무대. 시간표·학기·시험이 콘텐츠 리듬을 만든다. NPC와 학우가 살아있는 공간.

## 2. 레퍼런스 분석 — 무엇을 가져오고 무엇을 버리나

| 요소 | 아르피아에서 | 메이플에서 | 산나비에서 | 본작의 채택 |
|---|---|---|---|---|
| 성장 | 수업·퀘스트로 스킬/마법 습득, 종족·직업 | 레벨·SP·스킬트리·전직 | (선형 강화) | **수업/퀘스트 습득 + 레벨/SP 스킬트리 하이브리드** |
| 전투 조작 | 단순 | 사이드뷰 점프+스킬, 스타일리시 | 그래플·모멘텀 액션 | **메이플식 스킬 전투 + 산나비식 기동성(대시·그래플) 일부** |
| 타격감 | 약함 | 화려한 이펙트, 데미지 넘버 | 히트스톱·연출·카메라워크 | **둘의 장점 결합 (juice 시스템)** |
| 월드 | 학교 중심 커뮤니티 | 광활한 필드·마을 | 선형 스테이지 | **학교 허브 + 사이드뷰 필드/던전 챕터** |
| 진행 | 학년/커리큘럼 | 레벨 구간별 사냥터 | 스테이지 클리어 | **학기/학년 커리큘럼을 레벨 게이팅에 매핑** |

**핵심 방향 확정:**
- 겉모습·성장 구조·정체성 = **아르피아** (마법학교, 종족, 직업, 수업으로 배우는 마법)
- 조작·전투·타격감 = **메이플 + 산나비** (사이드뷰 액션, 스킬 연출, 기동성)

## 3. 세계관 컨테이너 (스토리 없이 구조만)

> 서사는 나중에. 지금은 스토리를 담을 **지리적·제도적 구조**만 확정.

### 3.1 무대: 마법학교 "아르카나 아카데미"
- 대륙 어딘가, 마력의 결정(크리스탈)을 중심으로 세워진 종합 마법 교육기관.
- 학교는 **거대한 부유 캠퍼스** 혹은 산맥에 걸친 성채 구조 — 여러 구역이 사이드뷰 맵으로 연결.
- 학생은 **입학 → 학년 진급 → 졸업(엔드게임)** 흐름으로 성장.

### 3.2 허브 구역 (플레이어가 상시 오가는 안전지대)
| 구역 | 기능 |
|---|---|
| 중앙 광장 | 스폰/집결지, 게시판(퀘스트·공지), 이벤트 |
| 강의동 (교실) | 수업 = 스킬 습득 콘텐츠 진입점 |
| 기숙사 | 세이브·휴식·인벤토리 정리·꾸미기(하우징 확장) |
| 훈련장 | 스킬 테스트·허수아비 DPS 체크·연습 |
| 도서관 | 스킬/마법 도감, 레시피, 로어, 연구(제작) |
| 식당/매점 | 소모품 상점, 버프 음식 |
| 마력공방 | 장비 제작·강화·인챈트 |
| 대회장/투기장 | 시험·랭킹·PvP(후순위) |
| 관문(게이트) | 각 필드/던전 챕터로 나가는 포탈 |
| 금지구역 입구 | 고난도 엔드 콘텐츠 게이트 |

### 3.3 필드 챕터 (전투가 벌어지는 사이드뷰 지역)
학년/레벨 구간별로 열리는 사냥터 & 스토리 무대. 예시 뼈대:
1. **입문의 숲** (Lv 1–10) — 튜토리얼 몬스터, 슬라임·정령
2. **속삭이는 지하서고** (Lv 10–20) — 언데드·마도서 몬스터
3. **원소 온실 / 폭풍 첨탑** (Lv 20–35) — 원소 몬스터, 속성 상성 도입
4. **뒤틀린 연금탑** (Lv 35–50) — 연금 골렘, 기믹 던전
5. **금지된 심연** (Lv 50+) — 엔드게임, 레이드 보스

각 챕터 = 여러 사이드뷰 맵 + 미니보스 + 챕터 보스 + 전용 퀘스트라인.

### 3.4 맵 그래프 & 이동 (v0.3 신설) ★
학교와 필드는 **여러 개의 사이드뷰 맵**으로 나뉘고, 맵 끝의 **포탈(관문/문/계단)**으로 연결된다 — 메이플의 맵 이동 방식.

```
[기숙사] ─ [캠퍼스 광장·강의동·훈련장(허브)] ─ 관문 ─ [입문의 숲 1] ─ [입문의 숲 2] ─ [시험장]
                     │           │                            │
                [도서관/공방]  [지하 계단]              [숨겨진 샛길(비밀맵)]
                                  │
                          [학교 지하실 (으스스)]
```

**지하실 (Basement) — 아르피아식 으스스한 미로 (v0.4 신설)**
- 캠퍼스에서 "수상한 계단"으로 진입. **어둡고 눅눅한 분위기**: 벽돌 벽, 벽 횃불, 떠다니는 먼지, 화면 가장자리 비네트(시야 제한 느낌).
- **긴 계단**으로 내려가며, 아래는 층층이 갈라지는 **미로형 발판**(상·하 분기, 막다른 곳)으로 살짝 복잡한 길.
- 서식: 그림자 박쥐 등 약한 유령형 몬스터(전투보단 분위기).
- **의문의 문(Mysterious Door):** 지하 깊은 곳의 굳게 잠긴 문. 상호작용(E)하면 잠긴 채 불길한 룬이 맥동하는 연출과 대사만 — **미래 콘텐츠 훅**(자격/열쇠/스토리 진행 시 개방 예정).
- **포탈 이동:** 포탈 앞에서 ↑(또는 E) → 짧은 전환 연출 → 다음 맵. 요구조건(레벨·랭크·퀘스트)이 걸린 포탈 존재.
- **허브 = 학교 캠퍼스:** 교사(수업), 훈련장(허수아비), 게시판(퀘스트)이 모두 이곳에. 필드는 전투 전용.
- 몬스터는 맵 입장 시 리스폰(보스 등 일회성 제외). 저장 데이터에 현재 맵 기록 → 이어하기 시 해당 맵에서 시작.
- 확장 원칙: 새 콘텐츠 = 새 맵 추가 + 포탈 연결(데이터 주도, §15.3).

### 3.5 월드 & 맵 확장 설계 (v0.5 신설) ★ — 학교다운 다층 캠퍼스 + 마을·항구·연무관

> 목표: 지금의 단조로운 야외 맵을 **아르피아처럼 "이어진 여러 공간"**으로 확장.
> ① 본관을 **여러 층(계단 이동)**으로, ② 학교 시설(교장실·양호실·동아리실·교실·매점)을 실제 공간으로,
> ③ **옆 마을·항구**로 나가는 경유 맵들, ④ **연무관(콜로세움)** 대결 콘텐츠까지.

        │                    │(입구)                             │

                                                                 ▼

                              │(별관)            │(관문)             │(뒷문/마을길)        │
                        [동아리 별관]      [입문의 숲]…[시험장]   [숲길]                   [훈련장]

- **간선(수업/성장):** 지하실 · 연무관 · 숲(전투) 가지, 마을·항구(생활/미래) 가지.

#### 3.5.3 야외 캠퍼스 (기존 campus 확장)
- 중앙 광장(분수·게시판), **본관 정문**(신설, →1F), **동아리 별관**(신설), 훈련장(허수아비), 관문(→숲), **마을길 뒷문**(신설, →숲길).
- **동아리실:** 동아리 NPC — 친목/서브 퀘스트, 코스메틱·소소한 버프, (미래)하우징 훅.

#### 3.5.4 연무관 (演武館 / 콜로세움) — 대결 콘텐츠 ★
아르피아의 콜로세움 오마주. 본관 1F에서 진입.
- **대기실:** 접수 NPC(모드 선택), **랭킹판**(최고 라운드·클리어 타임), 회복대.
- **투기장:** 원형 경기장 + 관중석 배경, 입장 시 문 봉쇄 → 대결 → 결과.
- **모드:**
  1. **대련(1:1):** 난이도별 NPC 도전자와 3라운드. 도전자 = 다른 마법사(원거리/근접 패턴).
  2. **연무 시험(웨이브):** 몰려오는 적을 라운드로 격파(점증 난이도) — 기존 몬스터·미니보스 재활용.
  3. **(미래) 순위전/친구 대결(PvP).**
- **보상:** 공적치/명예 토큰(전용 상점 재화), 종합 랭크(§6.1.6) 반영, 칭호.

#### 3.5.5 마을 가는 길 (경유 필드 맵)
"바로 마을"이 아니라 **여러 맵을 거쳐** 가도록(아르피아식 이동감).
| 맵 | 성격 | 요소 |
|---|---|---|
| **숲길** | 학교 뒷숲, 초반 | 약한 몬스터, 갈림길 샛장면 |
| **언덕 고갯길** | 전망·중반 | 조금 강한 몬스터, 절벽 발판, 미니 이벤트 |
| **마을 성문** | 관문 | 문지기 NPC(입장 대사), 마을 진입 |

#### 3.5.6 여명 마을 (옆 마을)
- **마을 광장:** 분수, **게시판(마을 의뢰)**, **여관**(세이브·휴식·전량 회복).
- **상점가:** 장비상·잡화상, (미래)대장간(제작·강화).
- **항구길 입구**(→소금바람 항구).

#### 3.5.7 소금바람 항구 (Harbor)

- **미래 관문:** 배를 타고 다른 지역(신규 대륙/해상 던전)으로 — 지금은 "출항 준비 중" 선원 NPC + 분위기 연출(항구 자체는 완성, 그 너머는 훅).

## 4. 종족 시스템 (Races)

아르피아처럼 종족이 정체성과 기본 성향을 결정. 종족은 **스탯 성향 + 종족 패시브 + 종족 전용 유틸 스킬 1개**로 차별화. 조작감에 실제 영향을 준다.

| 종족 | 콘셉트 | 스탯 성향(시작 랭크 §6.1.8) | 종족 패시브 | 종족 전용 스킬(예시) |
|---|---|---|---|---|
| **휴먼 (Human)** | 균형·적응 | 올라운드, 성장 보정 | 경험치·숙련도 획득 +5% | 「순간집중」: 짧은 캐스팅 가속 |
| **엘프 (Elf/정령족)** | 마력·민첩 | 마력·민첩 랭크↑, 체력↓ | 마나 회복 속도↑, 이동 속도↑ | 「정령 도약」: 공중 대시 1회 추가 |
| **드워프/노움 (Artificer)** | 제작·기술 | 방어·제작 특화 | 제작 성공률·강화 보정↑ | 「기계 설치」: 자동포탑 소환 |
| **수인 (Beastkin)** | 근접·기동 | 근력·민첩 랭크↑, 근접 화력↑ | 대시 쿨↓, 낙하 데미지 무효 | 「야성 질주」: 벽 타기/이단 벽점프 |
| **마족 (Umbral)** | 고위험 고보상 | 마력↑, 방어↓ | 저HP일수록 데미지↑(광폭) | 「그림자 잠행」: 짧은 은신+기습 크리 |

> **밸런스 원칙:** 종족은 "다른 재미"를 주되 "정답 종족"이 되지 않도록. 종족 보너스는 총 전투력의 5~10% 수준으로 제한하고, 나머지는 학과·빌드로 메운다.

> 초반 확장 여지: 각 종족에 소규모 시작 스토리·전용 첫 퀘스트 훅을 걸 수 있도록 구조만 열어둠(서사는 후속).

## 5. 학과 = 직업 시스템 (Classes / Disciplines)

아르피아의 "직업"을 **마법학교의 학과(전공)**로 번역. 종족과 독립적으로 선택(일부 조합에 소소한 시너지).

### 5.1 학과 진로 구조 (전직 트리)
```
[예과: 견습생] Lv 1–9  (공통 기본기)
        │  1차 학과 선택 (Lv 10)
        ▼
[1차 전공] ───────────────┐
   ├ 화염술사              │  2차 심화 (Lv 30)
   ├ 서리술사              ▼
   ├ 뇌전술사        [2차 마스터리 / 세부 특화]
   ├ 수호술사(방벽)   예) 화염술사 → 「업화술사(폭발/광역)」 or 「불꽃검사(근접 마검)」
   ├ 소환술사              │  3차 각성 (Lv 50, 엔드)
   ├ 마검사(근접)          ▼
   └ 치유술사(서포터)  [3차 각성: 학파의 대가]
```

### 5.2 1차 학과 6종 (MVP 목표)
각 학과는 **주 자원(마나/기력) + 조작 스타일 + 핵심 메커닉**이 다르다.

| 학과 | 역할 | 조작/전투 스타일 | 핵심 메커닉 | 대표 스킬 방향 |
|---|---|---|---|---|
| **화염술사 (Pyro)** | 원거리 딜러 | 차징·폭발형, 한 방 화력 | "과열" 스택 → 폭발 강화, 과열 시 자해 리스크 | 화염구 차지, 메테오, 화염 장판 |
| **서리술사 (Cryo)** | 컨트롤 딜러 | 견제·슬로우, 안전 딜 | "빙결" 스택 → 결빙(무력화) + 결빙 대상 추가딜 | 얼음창 관통, 서리 폭풍, 결빙 |
| **뇌전술사 (Volt)** | 기동 딜러 | 초고속 대시·연타, 콤보 | "감전" 연쇄(체인) + 대시 리셋 | 순간이동 찌르기, 연쇄 번개, 낙뢰 |
| **마검사 (Spellblade)** | 근접 브루저 | 메이플 근접+마법 부여 검격 | "마검 인챈트": 근접타에 속성 부여, 충전타 | 마력참, 돌진 베기, 오라 블레이드 |
| **소환술사 (Summoner)** | 펫/설치 | 소환수 운용, 진영 구축 | 소환수 지속·명령, 링크 강화 | 정령 소환, 골렘, 소환수 폭발 |
| **치유술사 (Mender)** | 서포터/힐러 | 지원·버프·소량 딜 | "은총" 게이지로 힐/버프/부활 | 치유, 보호막, 정화, 공격 버프 |

> 솔로 플레이도 성립하도록 각 학과에 최소한의 자립 딜·생존기 보장. 서포터도 혼자 사냥 가능(느리지만).

### 5.3 학과 선택 & 리스펙
- Lv 10 도달 + "학과 배정 시험" 퀘스트 클리어 시 1차 전공 확정.
- 각 학과 체험 퀘스트를 미리 플레이해보고 결정(맛보기).
- **리스펙(재분배):** SP 초기화는 유료/희귀 아이템, 학과 자체 변경은 상위 티어 아이템 or 특정 NPC 이벤트로 제한적 허용(정체성 유지).

## 6. 스탯 · 레벨 · 성장 시스템

### 6.1 스탯 랭크 시스템 (D ~ SSS) ★ v0.2 신규

> 『퀘스트지상주의』에서 영감. 플레이어에게 스탯은 **숫자가 아니라 "랭크(등급)"로만 보인다.**
> "근력 37" 대신 **"근력 C+"**. 성장의 재미를 미세한 숫자 증가가 아니라 **랭크가 오르는 순간의 쾌감**으로 전달한다. AP 자유 분배는 폐기.

#### 6.1.1 랭크 사다리
```
D-  D  D+  →  C-  C  C+  →  B-  B  B+  →  A-  A  A+  →  S  →  SS  →  SSS
```
- 총 **15단계**. 신입생은 대부분 **D-~D+** 구간에서 시작. (설정상 일반인은 E~F 영역이지만 게임 화면에는 미표기 — "입학생은 이미 재능이 검증된 자"라는 뉘앙스.)
- 세부등급(±)은 **A+까지의 잦은 성장 체감용**. S 이상은 ± 없이 희소성을 강조 — S가 뜨는 것 자체가 사건.
- **랭크 색상 언어:** D 회색 → C 초록 → B 파랑 → A 보라 → S 금색 → SS 진홍금 → SSS 프리즘(무지개 발광). UI·이펙트·장비 등급 색과 통일된 색 문법.
- 각 랭크 내부에는 숨겨진 **수련치 게이지**가 있고 가득 차면 승급. 게이지 표시도 % 숫자가 아니라 **별/구슬 눈금**(예: ◆◆◆◇◇)으로 — 숫자 비노출 원칙(§6.1.7) 준수.

#### 6.1.2 스탯 5종
| 스탯 | 지배 영역 |
|---|---|
| **근력 (STR)** | 근접·물리 데미지, 경직 유발력, 중장비 요구치 |
| **마력 (MAG)** | 마법 데미지, 최대 MP, 차지 속도 |
| **민첩 (AGI)** | 이동/공격 속도, 대시 쿨다운, 크리티컬 확률 |
| **체력 (VIT)** | 최대 HP, 방어, 경직 저항(슈퍼아머 게이지) |
| **정신 (WIL)** | MP 회복, 상태이상 저항, 캐스팅 안정(피격 시 차지 유지) |

- 학과별 주 스탯: 화염/서리/뇌전/소환/치유 = **마력**, 마검사 = **근력+마력**. 민첩·체력·정신은 전 학과 공용 보조축.

#### 6.1.3 랭크는 어떻게 오르나 (수련치 획득 경로)
| 경로 | 비중 | 설명 |
|---|---|---|
| **퀘스트 보상** | ★★★ 주력 | 퀘스트 완료 보상에 스탯 수련치가 명시된다. 웹툰처럼 **"보상: 근력 수련치 [대]"** 식 표기 — 크기는 [소]/[중]/[대]/[특대] 4단계 |
| **수업·훈련** | ★★★ 주력 | 수업(§7)·훈련장 미니게임이 관련 스탯을 단련. 마법이론→마력/정신, 비행·체술 수업→민첩, 실전 훈련→근력/체력 |
| **시험 성적** | ★★ | 학기말 시험·승급 시험의 평가 등급에 따라 보너스 수련치 |
| **행동 성장** | ★ 소량 | 실제 플레이가 스탯을 미세하게 단련: 대시/점프→민첩, 마법 적중→마력, 피격 후 생존→체력. **일일 상한**으로 노가다 방지 |
| **레벨업** | 자동 | 레벨업 시 학과 주 스탯에 수련치 보너스(자동 지급, 분배 없음) |
| **영약·비전서** | 희귀 | 희귀 소비 아이템 = 수련치 대량 or 즉시 세부등급 +1 (엔드 파밍 보상) |

> 설계 의도: **"강해지려면 학교 생활(퀘스트·수업)을 해야 한다"** — 사냥만 돌아서는 랭크가 잘 안 오른다. 필러 2(배우며 강해지는 성장)를 스탯 차원에서 강제하는 장치.

#### 6.1.4 승급 시험 (티어 돌파) ★
- 같은 문자 안의 ± 승급(D-→D→D+)은 수련치만 차면 **자동 승급**.
- **문자 티어 돌파**(D+→C-, C+→B-, B+→A-, A+→S, S→SS, SS→SSS)는 수련치 만충 + **승급 시험 통과**가 필요하다:
  - **근력:** 마력 측정기 타격 — 제한시간 내 파괴/최대 위력 측정
  - **마력:** 마력 결정 과부하 테스트 — 차지를 흔들림 없이 유지
  - **민첩:** 마법 장애물 코스 타임어택
  - **체력:** 시련의 방 — 장판·탄막 속에서 일정 시간 생존
  - **정신:** 환영 저항 시험 — 교란 속 집중 유지 미니게임
- 상위 돌파(A+→S 이상)는 시험 + **특수 조건**(희귀 재료, 전용 퀘스트라인, 교수 추천). **SSS는 사실상 엔드게임 위업** — 서버/저장 파일에 몇 명 없는 수준의 목표.
- **승급 연출:** 화면 정지 → 랭크 문자가 크게 각인되며 균열 → 새 색상으로 발광 → 캐릭터 오라 갱신. "한계를 돌파했다"는 체감이 이 시스템의 존재 이유.

#### 6.1.5 랭크가 주는 것 (효과 설계)
- 내부적으로 랭크마다 수치 구간이 매핑된 **디자인 테이블**이 존재(밸런싱은 숫자로, 표현은 랭크로).
- 랭크업 효과는 **계단식 점프**(문자 승급 시 +8~15% 구간 상승, ± 승급은 소폭)로 설계 — 미세 성장 대신 "확 강해지는" 순간을 만든다.
- **요구조건 게이트:** 장비·스킬·콘텐츠가 랭크를 요구한다. 예)
  - "메테오 습득 조건: **마력 B 이상** + 3학년 수업 이수"
  - "대마검사 로브 착용: **근력 C+**"
  - "금지구역 입장 허가: **종합 랭크 A- 이상**"
- 문자 승급 = 큰 스탯 점프 + 신규 해금이 동시에 터지는 이벤트 → 승급 시험에 도전할 강한 동기.

#### 6.1.6 종합 랭크 (학생 평가)
- 5개 스탯 + 레벨 + 시험 성적을 종합한 **학생 종합 랭크**(D~SSS)가 학생증에 표기된다.
- 용도: 학년 랭킹 게시판, NPC 반응 변화(D랭크 무시 → A랭크 존경), 일부 퀘스트/시설 입장 조건, 엔드 콘텐츠 자격.
- "학교에서 평가받는 학생"이라는 테마 장치이자, 5스탯을 고루 올릴 이유를 주는 장기 목표.

#### 6.1.7 숫자 비노출 원칙 (UI 규칙)
- 캐릭터 창에는 **랭크만** 표기. 공격력·방어력 등 파생 수치도 기본 비노출 — 장비 비교는 ▲▼ 화살표·랭크 영향 요약으로 표현.
- 단, **전투 피드백 숫자는 유지**: HP/MP 바, 데미지 넘버는 타격감의 필수 요소이므로 그대로. 비노출 대상은 "성장 스탯" 계열에 한정.
- 내부 시뮬레이션은 전부 정수 수치로 동작(치트·밸런스 테이블 별도 관리) — 랭크는 어디까지나 **표현 레이어**.

#### 6.1.8 종족별 시작 랭크 (예시)
| 종족 | 근력 | 마력 | 민첩 | 체력 | 정신 | 비고 |
|---|---|---|---|---|---|---|
| 휴먼 | D | D | D | D | D | 전 스탯 수련치 획득 +5% |
| 엘프 | D- | **D+** | **D+** | D- | D | |
| 드워프 | **D+** | D- | D- | **D+** | D | |
| 수인 | **D+** | D- | **D+** | D | D- | |
| 마족 | D- | **D+** | D | D- | **D+** | |

> 시작 랭크 차이는 초반 맛 차이일 뿐, 모든 종족이 모든 스탯을 SSS까지 올릴 수 있다(상한 차별 없음 — "노력이 재능을 이긴다" 테마).

### 6.2 레벨 & 경험치
- 레벨 상한(1차 목표): **Lv 60** (예과 1–9 / 1차 10–29 / 2차 30–49 / 3차 각성 50–60).
- 경험치 획득: 몬스터 처치, 퀘스트, **수업 이수(대량)**, 시험 통과, 채집/제작 보조.
- 레벨 = 콘텐츠 게이트(챕터·수업·장비 착용 요구 레벨).

### 6.3 성장 3축 (핵심 차별점)
성장은 서로 다른 세 축으로 굴러간다 — 각 축이 다른 재미를 담당:
1. **레벨 (진행 게이트)** — 콘텐츠·학년·장비의 열쇠. 사냥/퀘스트 경험치로 상승.
2. **스탯 랭크 (능력, §6.1)** — 퀘스트·수업·승급 시험으로 D→SSS. "내 캐릭터의 몸"이 강해지는 축.
3. **스킬 (전투 표현)** — 아래 두 경로의 결합:
   - **SP 스킬트리 (메이플식)** — 레벨업 시 SP 획득 → 스킬 레벨업/해금.
   - **습득(Learning) 시스템 (아르피아식)** — SP만으로는 못 배우는 스킬이 있다. **수업 수강·퀘스트 완료·마도서 해금**을 통해 스킬 슬롯 자체가 "해방"된다.

> 즉, 어떤 마법은 "레벨과 SP가 있어도" 해당 **수업을 듣지 않으면 배울 수 없다.** 이것이 학교 테마의 핵심 성장 루프.

### 6.4 숙련도(Mastery) 시스템
- 스킬을 많이 쓸수록 숙련도 상승 → 소폭 성능 향상 or 상위 변형 해금.
- 채집/제작/요리 등 생활 숙련도 별도 트랙(휴먼·드워프 유리).

## 7. 마법·스킬 습득 시스템 (수업 & 퀘스트) — ★ 게임의 심장

아르피아 감성의 핵심. "학교에서 마법을 배운다"를 실제 게임 루프로 구현.

### 7.0 스킬 제로 스타트 원칙 (v0.3 신설) ★
- **입학 직후 캐릭터는 스킬을 하나도 가지고 있지 않다.** 기본 공격(불티)조차 첫 수업에서 배워야 한다.
- 의도: "마법을 배운다"는 판타지를 첫 1분부터 체감시킨다. 아무것도 못 쏘던 캐릭터가 첫 수업 후 처음 불티를 쏘는 순간이 첫 번째 보상.
- 습득 전 스킬은 핫바에 **자물쇠**로 표시 — 무엇을 배우게 될지 미리 보여줘 기대감 형성.
- 수업 중에는 해당 스킬이 **임시 각인**되어 과제 수행 가능, 과제 완료 시 **정식 습득**(영구).

### 7.5 교사 티어 시스템 (v0.3 신설) ★
학과마다 **초급 → 중급 → 고급 → 대마법사** 교사가 존재. 상위 교사일수록 **화려하고, 강력하고, 범위가 넓은** 마법을 가르친다 — 대신 자원 소모·쿨다운도 커진다(운영의 깊이).

| 티어 | 교사 (화염술) | 해금 조건 | 가르치는 스킬 성향 |
|---|---|---|---|
| **초급** | 아그니스 교수 | 입학 즉시 | 기본기 — 단일 대상, 저비용, 짧은 쿨 |
| **중급** | 이그니아 교수 | **1학기 학기말 시험(보스) 합격** | 관통·광역 진입 — 다수 타격, 중간 비용 |
| **고급** | (미공개) | 중급 커리큘럼 이수 + 마력 랭크 B- 이상 + 2학기 시험 | 대규모 광역·소환 — 화면을 뒤덮는 화력, 긴 쿨 |
| **대마법사** | (엔드게임) | 3차 각성 + 특수 퀘스트라인 | 궁극기 — 전투를 바꾸는 한 방 |

**화염술사 커리큘럼 예시** (다른 학과도 동일 구조):
| 티어 | 스킬 | 키 | 타입 | 특징 |
|---|---|---|---|---|
| 초급 1 | 불티 | Z | 기본탄 | 마나 무소모, 빠른 연사 — 첫 수업 |
| 초급 2 | 화염구 | X | 차지탄 | 차지할수록 크기·폭발↑ |
| 초급 3 | 화염 장판 | C | 설치 | 지속 도트, 유인 플레이 |
| 중급 1 | 플레임 랜스 | V | 관통 | 일직선 다수 관통, 고속 |
| 중급 2 | 폭염 노바 | S | 광역 폭발 | 자기 중심 360° 폭발+넉백, 위기 탈출 |
| 고급 1 | 메테오 | Q | 궁극 광역 | 낙하 대폭발, 최대 화력 |
| 고급 2 | 불사조 소환 | (예정) | 소환 | 지속 화력 지원 |
| 고급 3 | 지옥불 지대 | (예정) | 초광역 설치 | 화면 절반을 불태움 |

- 교사는 캠퍼스 강의동에 상주. 해금 전 상위 교사는 실루엣/잠금으로 보여 **다음 목표를 시각화**.
- 각 교사 = 수업 체인(2~4개). 수업마다 실습 과제(§7.1 형태) → 통과 시 스킬 정식 습득 + 수련치 보상.
- 티어 진급 = 학기 진행(§7.2)·시험(보스전)·스탯 랭크(§6.1) 조건이 맞물리는 구조 — 세 성장축의 교차점.

### 7.1 수업(Class/Lecture) 콘텐츠
수업은 강의동에서 진입하는 **미니 콘텐츠**. 단순 텍스트가 아니라 **플레이 가능한 실습**.

수업 1회 = 아래 중 하나의 형태:
- **강의 실습형:** 새 스킬을 받고, 지정 표적/시나리오에서 조건 달성(예: "화염구로 허수아비 5개를 3초 내 격파").
- **필드 실습형:** 지정 필드에서 특정 몬스터를 새 스킬로 처치.
- **이론 시험형:** 마법 원리 퀴즈/상성 맞히기(가벼운 두뇌 콘텐츠, 소량 보상).
- **실전 평가형:** 미니보스전 or 타임어택.

수업 보상: **신규 스킬 해금(습득 슬롯 개방)** + 경험치 + 소액 재화 + 숙련.

### 7.2 커리큘럼 구조 (학기제 = 콘텐츠 리듬)
```
1학년
 ├ 1학기: 기초 마법이론 → 견습 기본기 (이동/캐스팅/기초 원소)
 └ 2학기: 원소 개론 → 1차 학과 선택 & 학과 기초 스킬
2학년
 ├ 1학기: 전공 심화 → 2차 스킬군, 상성/콤보
 └ 2학기: 응용 마법 → 유틸/기동/생존기
3학년
 └ 각성/졸업연구 → 3차 각성기, 캡스톤(챕터 보스 처치)
```
- 각 학기 종료 = **학기말 시험(보스전/평가)** → 통과 시 진급 & 다음 챕터 개방.
- 수업은 필수(메인)와 선택(서브·부가 스킬/생활) 구분.

### 7.3 스킬 해금 3경로
1. **정규 수업 이수** (메인 스킬)
2. **퀘스트/이벤트 보상** (히든·특성 스킬)
3. **마도서/스크롤 아이템** (드랍·제작·상점 — 부가·변형 스킬)

### 7.4 스킬 편성(빌드)
- **액티브 스킬 슬롯:** 키 바인딩 가능한 액티브 6~8개(핫바).
- **패시브:** 상시 적용.
- **각인/룬(선택 심화):** 스킬 하나에 룬을 끼워 특성 변경(예: 화염구 → 관통형/분열형/추적형). 빌드 다양성의 후반 축.

### 7.6 스킬 강화 시스템 (v0.6 신설 · 구현됨) ★
**원칙: "습득(배움)"과 "강화(숙련)"는 별개의 성장 축.** 수업으로 스킬을 배우고(§7.0~7.5), 강화로 그 스킬을 깊게 단련한다.

#### 7.6.1 레벨 구조
- 스킬별 **레벨 1~5 (MAX)**. 습득 시 Lv.1.
- **데미지: 레벨당 +25%** (Lv.5 = 기본의 200%).
- 스킬별 부가 성장 (구현 수치):

| 스킬 | 레벨당 추가 성장 |
|---|---|
| 불티 (Z) | 탄 크기 +1 · 탄속 +30 · 사거리 +30 · 넉백 +10 |
| 화염구 (X) | **폭발 반경 +13%** · 탄 크기 +10% · 화면 흔들림 +8% |
| 화염 장판 (C) | **폭 +26px · 지속 +0.4초** · 불기둥 높이↑ |
| 플레임 랜스 (V) | 사거리 +50 · 탄속 +40 · 두께 +1.5 · 넉백 +15 |
| 폭염 노바 (S) | **반경 +20** · 파티클/충격파↑ |

#### 7.6.2 이펙트 티어 — "강해짐이 눈에 보인다"
| 티어 | 레벨 | 연출 |
|---|---|---|
| T0 | 1~2 | 기본 주황 화염 |
| T1 | 3~4 | 글로우 확대 + **이중 폭발 링** + 파티클 증가, 밝은 금빛 |
| T2 | **5 (MAX)** | ★ **푸른 화염 각성** — 투사체·폭발·장판·트레일·시전 이펙트가 전부 청염(靑焰)으로 변화 |

> 시각 언어 규칙: 성능 상승은 반드시 연출 상승을 동반한다. 다른 학과(빙결·뇌전 등) 확장 시에도 "MAX = 색 각성" 문법을 유지.

#### 7.6.3 강화 경로
- **[현재 프로토]** 교정 훈련장의 **훈련 교관** NPC — E로 강화 패널 → 숫자 [1~5] 키로 해당 스킬 즉시 강화 (무료·편의 기능임을 UI에 명시).
- **[목표 설계]** 아래 경로들이 교관 UI 위에 비용/조건으로 얹힌다:
  1. **심화 수업 이수** — 같은 스킬의 상위 수업(예: 「불티 심화」)을 통과하면 +1 Lv.
  2. **스토리/퀘스트 보상** — 챕터 진행·히든 퀘스트로 특정 스킬 강화권.
  3. **스킬북 아이템** — 보스 드랍·제작·상점.
  4. **재화 소모** — 골드 + 스탯 수련치 일부를 비용으로(성장 축 간 상호작용).
- 제한: 미습득 스킬은 강화 불가, MAX 초과 불가. 강화 상태는 저장 데이터에 보존.

#### 7.6.4 밸런스 공식
```
최종 데미지 = 스킬 기본치 × 마력 랭크 배율(§6.1, +6%/랭크) × 스킬 레벨 배율(+25%/Lv) × 크리티컬(×1.5)
```
- 스탯 랭크(수업·퀘스트로 성장)와 스킬 레벨(수련으로 성장)이 **곱연산**으로 겹침 → 두 축 모두 돌 이유가 생긴다.
- 범위 성장은 데미지와 분리된 별도 축 — "레벨을 올리면 플레이 감각 자체가 넓어진다".

### 7.7 스킬 도감 (v0.6 신설 · 구현됨) ★
- **진입:** 학생 수첩(Tab) → **←/→ 로 2페이지** `① 학생부 ↔ ② 스킬 도감` 전환.
- **표기 정보 (스킬별):**
  - 이름 · 단축키 · 현재 레벨(◆ 핍 5칸, MAX는 푸른색)
  - 한 줄 설명(플레이버 겸 기능 요약)
  - **실계산 데미지 범위** — 마력 랭크 × 스킬 레벨이 반영된 지금 이 캐릭터의 실제 수치
  - MP 소모 · 쿨타임 · 사거리/폭발반경/지속시간/특성(관통·무적 등)
  - 다음 레벨 미리보기 (+25% · 범위↑) / MAX는 "푸른 화염 각성 완료"
- **미습득 스킬:** 잠금 표시 + "수업 이수 후 정보가 공개됩니다" — 배울 것이 보여 기대감 형성(§7.0의 자물쇠 핫바와 동일 문법).
- **§6.1.7(숫자 비노출)과의 관계:** 스탯은 랭크로 숨기지만, **스킬 성능은 도감에서 명시적 공개** — "내 마법을 연구한다"는 도감의 존재 이유이므로 예외로 확정.
- **확장 계획:** 몬스터 도감·아이템 도감 탭 추가, 도서관(사서 NPC) 연구 콘텐츠와 연동.

### 7.8 고급 마법 (v0.7 신설 · 1식 구현됨) ★
**철학: 고급 마법은 '수업 신청'으로 못 배운다.** 대마도사가 인정할 때까지의 **어려운 여정(퀘스트 체인)**을 거쳐 **하나씩** 습득한다. 위력은 궁극기급 — ①대인 초고화력(인페르노 드래곤형 램프업) 또는 ②화면을 뒤덮는 초광역(메이플 6차형).

#### 7.8.1 습득 여정 — 『홍염의 서』 퀘스트 체인 (구현됨)

|---|---|---|
| 1 | 금서의 소문 | 중급 과정(랜스+노바) 수료 시 3층의 "???"가 **발더스 대마도사**로 정체 공개 — 대면 |

#### 7.8.2 제1식 「인페르노 번」 (Q 홀드 — 구현됨)
- **채널링 광선**: Q를 유지하면 시선 방향의 가장 가까운 적에게 광선 고정. 이동 불가, MP 10 + **초당 13 지속 소모**, 해제 시 쿨 8초.
- **램프업**: 열기 게이지가 2.6초에 걸쳐 차오르며 틱 데미지가 **최대 3.6배**로 폭증 (0.15초당 1틱) — "오래 지질수록 무섭다".
- **연출**: 3중 레이어 광선(외곽·중간·코어) + 임팩트 글로우 + 열기 비례 파티클/셰이크, MAX HEAT 표시. **Lv.5 강화 시 청염 광선**.
- 피격 시 채널 강제 해제(쿨 적용) — 리스크 있는 궁극기.

#### 7.8.3 제2식 「멸화」 — 불의 마을 퀘스트 체인 (v0.13 구현됨) ★
**습득 여정**: 제1식을 완전히 습득하면 대마도사가 제2식 『멸화』를 언급하며, 그 불씨를 얻기 위해 **바다 건너 '불의 마을'**로 보낸다.

|---|---|---|
| 1 | 붉은 편지 | 대마도사에게 제2식에 대해 듣기 → **소금바람 항구**로 안내(배편 개방) |

| 3 | 화산의 시련 | 화산 정상의 미니보스 **「화염 군주 이그니스」**(HP 1050, 용암 갑주) 격파 |
| 4 | 멸화의 불씨 | 쓰러진 군주가 남긴 **『멸화의 불씨』** 회수(E) |

- 이후 고급 마법도 **별도의 어려운 퀘스트 체인**(새 지역·시련)으로 하나씩 개방하는 구조 유지.

#### 7.8.4 불의 마을 (신규 지역 · v0.13) ★

- **몬스터**: 화염 정령(부유형)·화염 군주 이그니스(퀘스트 미니보스).

### 7.9 수업 선택 창 (v0.13 신설 · 구현됨) ★
교수에게 **E**로 말을 걸면 **학생 수첩(책) 형태의 수업 선택 창**이 열린다. 그 교수가 가르치는 모든 술식이 **이수 완료 / 수업 진행 중 / 배우기 가능 / 잠김** 상태로 나열되며, ↑/↓(또는 클릭)로 고르고 **E/Enter**로 배운다.
- **선택 습득**: 한 교수의 여러 술식을 **원하는 순서로** 골라 배울 수 있다(선행 술식·퀘스트 조건이 있으면 '잠김'으로 사유 표시).
- **잠김 사유 안내**: 예) 멸화는 "불의 마을에서 『멸화의 불씨』를 가져와야 전수받을 수 있다".
- 선택 시 기존 수업 흐름(대사 → 임시 각인 → 과제 → 정식 습득)으로 이어진다.

### 7.10 메인 스토리 스파인 — 54 퀘스트 체인 (v0.14 신설 · 구현됨) ★★
**철학: "시뮬레이터"가 아니라 "게임".** 스킬 습득은 더 이상 단발 과제가 아니라, 튜토리얼부터 고급 마법까지 이어지는 **하나의 학교생활 스토리**의 일부다. 스킬 하나를 얻으려면 **이론 수업 + 심부름 퀘스트 + 과제 + 각인 실습**의 여러 단계를 거쳐야 한다.

#### 7.10.1 흐름 구조
| 구간 | 퀘스트 | 내용 |
|---|---|---|
| **튜토리얼** (0~5) | 첫 걸음~교문을 지나 | 신규 맵 '학교 가는 길'에서 이동·점프·대시·아래점프 학습, 동급생 **리나**와 첫 만남, 입학 |
| **오리엔테이션** (6~15) | 본관 로비~수료 | 담임 **아그니스** 대면 → 도서관·사서·양호실·매점·훈련실·교관·교장실 투어 (학교 구조를 스토리로 학습) |
| **불티 아크** (16~20) | 이론 I~각인 실습 | 마나 이론 수업 → 도서관에서 교과서 찾기 → **테오**의 필기 노트 → 교관의 마력 측정 → 각인 실습(10회 적중) |
| **화염구 아크** (21~25) | 리나의 부탁~실습 | 리나의 잃어버린 목걸이 찾기(숲) → 보답 정보 → 응축 이론 → 각인 실습(MAX 차지 3회) |
| **장판 아크** (26~31) | 숲 정리~실습 | 슬라임 5 → 테오의 버섯 공포증 → 버섯 소탕 → 감사 → 각인진 이론 → 각인 실습(20회 타격) |
| **학기말 시험** (32) | 실기 시험 | 숲의 수호자 격파 |
| **랜스 아크** (33~38) | 중급 입학~실습 | 이그니아 대면 → 교장의 추천서 → 여명 마을 소포 심부름 → 관통 이론 → 각인 실습(15회) |
| **노바 아크** (39~43) | 잡화점 부탁~실습 | 슬라임 젤리 5개 배달 → 파동 이론 → 각인 실습(12회) |
| **고급 1식** (44~48) | 『홍염의 서』 체인 | §7.8.1 그대로 |
| **고급 2식** (49~53) | 불의 마을 체인 | §7.8.3 그대로 |

#### 7.10.2 스토리 시스템 요소
- **인트로 내레이션**: 신규 시작 시 입학 첫날 내레이션 → 튜토리얼 언덕에서 시작.
- **동급생 NPC**: 리나(활발·핑크)·테오(소심·파랑) — 교실 상주, 심부름 서브스토리 담당.
- **스토리 대화 시스템**: 퀘스트가 특정 인물(sid)과의 대화이면 그 인물이 **퀘스트 전용 대사**를 재생 (`handleStoryTalk`).
- **퀘스트 마커**: 지금 말을 걸어야 하는 인물 머리 위에 **금색 "!"** 표시.
- **수집 스팟**: 퀘스트 아이템(교과서·목걸이 등)은 맵의 **금빛 반짝임 지점**에서 E로 회수 (`spot` 데이터).
- **행동 퀘스트**: walk(이동 거리)·act(jump/dash/dropjump)·visit(맵 도착) 타입으로 튜토리얼·투어를 퀘스트화.
- **수업 게이트**: 각 스킬의 각인 실습은 해당 아크의 이론·심부름을 마쳐야 해금 (`req` + 잠김 사유 안내).
- **가벼운 톤**: 최종 빌런 서사는 후속 — 현재는 학교생활·부탁 들어주기 중심 (지하실 '의문의 문'과 교장의 암시가 후속 떡밥).

#### 7.10.3 구버전 세이브 마이그레이션
- 세이브에 `qv`(퀘스트 체인 버전) 필드 도입. `qv:2`면 그대로 신뢰.
- 구버전(qv 없음) 세이브는 **보유 스킬·보스 격파·불씨 소지 상태**로 신규 체인 위치를 자동 계산(`pd` 판정 워크) — 레벨·스탯·스킬·강화는 전부 유지.
- 구 고급 체인(8~17) 진행자는 단계 오프셋으로 정밀 보정 (예: 구 '화산의 시련' 진행 중 → 신 51번으로 복원).

### 8.2 기동성 (산나비 감성 일부 이식)
- **대시 캔슬:** 스킬 후딜을 대시로 캔슬 → 콤보 흐름 유지(고수용 심화).
- **에어 컨트롤:** 공중 스킬·공중 대시로 체공 딜.
- **종족/학과 기동기:** 벽점프(수인), 공중 대시(엘프), 순간이동(뇌전술사), 그래플 훅(특정 유틸 아이템/스킬)로 지형 활용.
- **모멘텀 보존:** 대시 후 이동 관성 소량 유지로 "미끄러지는" 손맛.

### 8.3 타격감(Juice) 시스템 — 기존 엔진 확장
Shadow Dagger에서 검증된 요소를 마법 전투로 확장:
1. **히트스톱** — 스킬 위력 비례 프레임 정지(막타·크리 강조).
2. **화면 흔들림** — trauma 기반, 폭발/광역기에 강하게.
3. **슬로우모션** — 보스 페이즈 전환·퍼펙트 회피·막타.
4. **파티클** — 원소별 전용(불티/서리결정/스파크/마력파편/소환진).
5. **글로우/애디티브 블렌드** — 마법 특유의 발광.
6. **콤보 시스템** — 연속 히트 카운터 → 데미지 증가 + 콤보 연출 + 등급(콤보 랭크).
7. **데미지 넘버** — 부유 숫자, 크리/속성 강조 색.
8. **줌 펀치** — 대형 타격 시 카메라 미세 줌인.
9. **속성 반응 연출** — 감전+빙결 등 상성 발동 시 전용 이펙트/추가 데미지.
10. **처치 연출** — 몬스터 사망 시 파편 분해 + 드랍 팝.

### 8.4 자원 시스템
- **마나(MP):** 마법 시전 소모, WIL/장비로 회복. 고갈 시 평타/기본기로 순환.
- **기력/오버차지 게이지:** 평타·피격·회피로 충전 → 궁극기/각성기.
- **학과별 서브 자원:** 과열(화염), 빙결스택(서리), 감전체인(뇌전), 은총(치유), 소환 유지비(소환) 등 — 학과 정체성 강화.

### 8.5 속성 상성 & 반응
- 속성: 화(火)·빙(氷)·뢰(雷)·물리·빛·어둠.
- 몬스터별 약점/저항 → 파티/빌드에 상성 고려 유도(강제 아님, 보너스형).
- **원소 반응(콤보):** 예) 빙결 대상 → 뇌전 시 "과전도"(추가 광역), 화염 대상 → 서리 시 "급랭"(경직) 등. 파티·멀티스킬 플레이의 깊이.

### 8.6 몬스터 & 보스 설계
- **일반 몬스터:** 명확한 windup(예열) → 공정한 회피 학습.
- **엘리트:** 필드 배회, 특수 패턴, 좋은 드랍.
- **미니보스/챕터보스:** 페이즈제(Shadow Knight식 3페이즈 계승), windup 플래시, 광역/장판/소환 패턴, 각 학과가 "다르게 공략"되도록.
- **레이드(엔드):** 다인 협동 기믹(후순위 로드맵).

## 15. 기술 아키텍처 (제안)

> 현 저장소는 의존성 0 단일 `index.html`(canvas+vanilla JS) 기조. 신작 스코프를 고려해 두 갈래 제안.

### 15.1 옵션 A — 기존 기조 유지 (프로토/웹 데모)
- 순수 `<canvas>` + vanilla JS, 단일/소수 HTML. 빠른 프로토·GitHub Pages 즉시 배포.
- 장점: 진입장벽 0, 현 프로젝트와 일관. 단점: RPG 규모(맵·인벤·저장) 확장 시 관리 부담.
- **권장:** 전투 손맛 프로토타입(수직 슬라이스)을 이 방식으로 먼저 검증.

### 15.2 옵션 B — 엔진 채택 (본 개발)
- **Godot(추천, 2D·경량·무료·픽셀친화)** 또는 Phaser(웹) / Unity(생태계).
- 장점: 타일맵·물리·애니·씬 관리·세이브·입력·오디오 내장 → RPG 규모에 적합.
- **권장 경로:** 프로토(옵션 A로 손맛 확정) → 본편(Godot로 이관).

### 15.3 공통 아키텍처 원칙
- 고정 타임스텝 루프(accumulator) + 렌더 보간.
- 데이터 주도 설계: 스킬/몬스터/아이템/수업/퀘스트를 **JSON/리소스 데이터**로 정의(밸런싱·확장 용이).
- 시스템 분리: Input / Physics·Collision / Combat / Skill / Stat·Progression / Quest / Inventory / Save·Load / UI / FX(juice) / SceneFlow.
- 세이브: 캐릭터·진행·인벤·퀘스트·스킬습득 상태 직렬화.
- 스킬 시스템은 **효과(effect) 컴포넌트 조합**(데미지/상태이상/이동/소환/버프)으로 데이터화 → 신규 스킬 추가 비용 최소화.

## 17. 리스크 & 미결 결정 사항 (검토 요청)

아래는 **당신의 결정이 필요한 지점** — 회신 주시면 다음 개정에 반영합니다.

1. **개발 방식:** 옵션 A(단일 HTML canvas 유지) vs 옵션 B(Godot 이관)? 프로토는 A로 가고 본편 B 이관을 권장.
2. **범위/스코프:** 1인/소규모 개발이면 종족·학과 수를 MVP에서 축소(예: 종족 2, 학과 3)할지?
3. **온라인성:** 순수 싱글? 아니면 파티·거래·랭킹 등 온라인 요소를 어디까지? (초기엔 싱글 권장)
4. **성장 상한/템포:** 만렙·학년 수·학기 길이 — 캐주얼(짧게) vs 코어(길게)?
5. **전투 무게중심:** 메이플식 "경쾌·다타격" 쪽 vs 산나비식 "묵직·저타격 고임팩트" 쪽 — 슬라이더 어디에?
6. **강화 스트레스:** 파괴/하락 있는 하드코어형 vs 완화형 캐주얼? (캐주얼 가정)
7. **아트 리소스:** 도트 에셋 자체 제작 vs 에셋 구매/AI 보조? 캐릭터 픽셀 사이즈(32/48/64)?
8. **가제 확정:** "아르카나 아카데미" 유지 여부, 학교/종족/학과 명칭 톤(한글식 vs 판타지식).
9. **스탯 랭크 세분화:** ± 포함 15단계안(현안) vs 단순 7단계(D~SSS)? 행동 성장(플레이로 미세 단련) 채택 여부, 승급 시험을 5스탯 전부 별도 미니게임으로 만들지(제작비) vs 공용 시험 1종으로 시작할지.
